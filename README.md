东方符斗祭
==========

简介
----------

东方符斗祭是一个以东方Project为背景类“三国杀”卡牌游戏。
原版（纯 Python 版）已不再维护，可以在 [legacy 分支](https://github.com/feisuzhu/thbattle/tree/legacy) 找到。
目前这里是符斗祭的核心代码，包括游戏逻辑、游戏服务端、后端以及部署相关的描述符。游戏客户端因为使用了商业插件无法开源。

仍然在开发中。
